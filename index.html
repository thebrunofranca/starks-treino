TREINAMENTO STARKS

Overview

O intuito deste projeto é desenvolver um webpage completo desde a estruturação inicial até sua personalização. Desse modo, individualmente, os novos membros da squad irão adquirir um conhecimento de todo o processo de desenvolvimento front-end - o que é importante para a gestão de conhecimento do time, visto que busca equilibrar o nível de conhecimento do mesmo.

AWESOME FEATURES

Tecnologias Utilizadas

JAVASCRIPT

É atualmente a principal linguagem para programação client-side em navegadores web.

HTML5

É uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS

É um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.

GIT

Tem a finalidade de gerenciar diferentes versões no desenvolvimento de um projeto. Uma das ferramentas mais famosas nesse meio é o Git, que possibilita o desenvolvedor versionar arquivos e agrupá-los com o desenvolvimento de outros integrantes do time.

SASS

É uma linguagem baseada em scripts que possibilita o desenvolvedor melhorar sua performance quanto ao layout de suas páginas web. O SASS possibilita utilizar lógicas de programação para estilização, compilando seu arquivo para um arquivo do tipo CSS (citado acima).

TYPESCRIPT

É um superconjunto de JavaScript que compila para JavaScript simples e adiciona tipagem e alguns outros recursos a linguagem.

BOOTSTRAP

É um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, que melhora a experiência do usuário em um site amigável e responsivo.

JQUERY

É uma biblioteca de funções JavaScript que interage com o HTML, desenvolvida para simplificar os scripts interpretados no navegador do cliente (client-side).

GULP

É um executor de tarefas que utiliza a plataforma Node.js para gerenciar a automação do processo de desenvolvimento web como fazer watch de pré-processadores.

JENKINS

Jenkins é um servidor de Integração Contínua open-source feito em Java, pode ser rodado de forma standalone ou como uma web aplicação dentro de um servidor web.

ADOBE EXPERIENCE MANAGER

É um sistema de gerenciamento de conteúdo (CMS - content management system) que provê um ambiente para criar, gerenciar e entregar experiências digitais em sites web e mobile.
